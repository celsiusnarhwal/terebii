# Changelog[^1]

Notable changes to Terebii are documented here.

Terebii adheres to [semantic versioning](https://semver.org/spec/v2.0.0.html).

## <a name="1-1-0">1.1.0 — 2026-01-24</a>

### Added

- The new `TEREBII_INCLUDE_DOWNLOADED` setting allows you to control whether Terebii will send notifications for
episodes that have already been downloaded.


## <a name="1-0-0">1.0.0 — 2026-1-12</a>

This is the initial release of Terebii.
